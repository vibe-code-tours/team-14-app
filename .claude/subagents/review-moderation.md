# Review Moderation Subagent

## Purpose

Protect the integrity of anonymous worker reviews.

---

## Use When

- Building review submission
- Review validation
- Spam prevention
- Useful voting
- Review moderation

---

## Responsibilities

- Anonymous reviews
- Review validation
- Spam detection
- Useful voting
- Moderation workflow

---

## Inputs

- Review payload
- User request
- Company information

---

## Outputs

- Valid review
- Moderation decision
- Updated review statistics

---

## Standards

Always

- Protect anonymity
- Validate input
- Sanitize content
- Prevent duplicate voting
- Record moderation actions

Never

- Reveal reviewer identity
- Delete reviews without audit
- Trust client input

---

## Definition of Done

- Review validated
- Review stored
- Statistics updated
- Privacy preserved

# Bug Fix Workflow

## Purpose

Provide a consistent process for fixing bugs.

## Process

1. Reproduce the bug.
2. Identify root cause.
3. Implement the fix.
4. Verify no regression.
5. Update tests if required.
6. Update documentation if behavior changed.
7. Merge.

## Definition of Done

- Bug reproduced.
- Root cause identified.
- Fix verified.
- Tests passed.
- No regression.

# Product Vision

Build a trusted platform where Myanmar migrant workers can safely research factories and agencies before changing jobs.

---

# MVP Goal

A worker can:

- Search a company
- View company information
- Read anonymous reviews
- Submit an anonymous review
- Vote Useful / Not Useful

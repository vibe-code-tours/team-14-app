# Release Checklist

## Before Release

- Feature Complete
- Pull Request Approved
- Code Review Completed
- Security Review Completed
- Tests Passed
- Build Passed
- Documentation Updated
- Database Migration Reviewed

## Deployment

- Backup Database
- Deploy Application
- Apply Database Migration
- Verify Health Check
- Verify APIs
- Verify UI

## After Release

- Monitor Logs
- Monitor Errors
- Verify Production
- Notify Team
- Update Release Notes
